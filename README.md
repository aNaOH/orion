# Orion
## Gamers, united!

Orion is a social network (to write)

# Requirements

- An Apache and PHP environment
- Composer ([Get it here](https://getcomposer.org/doc/00-intro.md))
- Cloudflare account and bucket

# How to install

1. [b]Clone this repository using the terminal[/b]

```
git clone https://github.com/aNaOH/orion.git
```

2. [b]Install Composer packages[/b]

Depending on the Composer installation, the method can be one of the following:

```
composer install
php composer.phar install
```
